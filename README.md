# bundler-benchmark

Benchmark some popular TS bundlers :
- [tsdown](https://tsdown.dev/)
- [tsup](https://tsup.egoist.dev/)
- [unbuild](https://github.com/unjs/unbuild)

### Development

- Install dependencies
```bash
npm install
```
- Generate the benchmark data
```bash
npm run generate
```
- Run the benchmark
```bash
npm run benchmark
```
